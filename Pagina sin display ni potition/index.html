Mis Proyectos

Proyecto 1

EcoTrack: Monitor de Huella de Carbono es un proyecto ideal para practicar lógica de programación. Es una aplicación donde el usuario ingresa sus hábitos diarios (transporte, consumo eléctrico, dieta) y la app calcula el impacto ambiental.

Proyecto 2

TaskMaster: Gestor de Tareas Semanal es la clásica "To-Do List" pero con un giro: permite organizar tareas por prioridades y categorías (Estudio, Trabajo, Personal) usando colores. Incluye una barra de progreso que se llena al completar tareas.

Ver proyecto

Proyecto 3

DevMatch: Buscador de Perfiles GitHub es una herramienta que utiliza la API de GitHub para buscar desarrolladores. Al ingresar un nombre de usuario, la tarjeta muestra su foto, biografía, número de repositorios y lenguajes más usados.

Ver proyecto

Proyecto 4

CineBox: Catálogo de Películas Favoritas es un sitio web donde puedes listar tus películas preferidas. Cada una tiene una tarjeta con su póster, una breve reseña y una calificación con estrellas. Incluye un filtro por género (Acción, Comedia, Drama).
